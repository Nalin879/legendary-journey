This project is a clean, static-first personal website that keeps content in simple markdown and MDX files.

## What it does

- Renders blog posts and project pages as static HTML
- Keeps content easy to edit and version control friendly
- Supports rich markdown with simple custom layouts

## Why it works well

The setup is intentionally minimal so the site feels calm, readable, and easy to maintain.
